# Test Action Style Guide

This is a test repository for [action-style-guide](https://github.com/QuantEcon/action-style-guide).

## Purpose

This Jupyter Book contains test lectures used to validate the QuantEcon Style Checker GitHub Action.

## Test Lectures

- **Clean Lecture**: A properly formatted lecture that should pass all style checks
- **Violations Lecture**: A lecture with intentional style violations for testing

## Usage

This repository is used for:

1. **Regression testing** - Verify the style checker detects known violations
2. **Development testing** - Test new rules before deploying to production
3. **CI/CD validation** - Ensure the GitHub Action works correctly

See the [action-style-guide repository](https://github.com/QuantEcon/action-style-guide) for documentation.
